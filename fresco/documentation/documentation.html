Installation

First make sure the HTML5 doctype is set as the very first line on the page:

<!DOCTYPE html>

Include jQuery 1.4.4 or above in the header. Going with the latest 1.10.x release is recommended for optimal performance, the jQuery CDN can be used for this.

Download Fresco and upload the files from the package to your server. Include fresco.js and its dependancies below jQuery. Also include fresco.css and make sure the include paths point to the locations where the files have been uploaded.

Including the Youtube Iframe API, is not required, adding it allows Fresco to force HD quality onto Youtube videos.

<!-- jQuery -->
<script type="text/javascript" src="http://code.jquery.com/jquery-1.10.2.min.js"></script>

<!-- Make IE8 and below responsive by adding CSS3 MediaQuery support -->
<!--[if lt IE 9]>
  <script type="text/javascript" src="http://css3-mediaqueries-js.googlecode.com/svn/trunk/css3-mediaqueries.js"></script>
<![endif]-->

<!-- Youtube and Vimeo API (improves video on touch devices and gives better HD support) -->
<script type="text/javascript" src="http://www.youtube.com/iframe_api"></script>
<script type="text/javascript" src="http://a.vimeocdn.com/js/froogaloop2.min.js"></script>

<!-- Fresco -->
<script type="text/javascript" src="/js/fresco/fresco.js"></script>
<link rel="stylesheet" type="text/css" href="/css/fresco/fresco.css"/>

That's it! Fresco should now be ready for use.

= Note: The downloads have an example folder demonstrating the installation. Leave a message on the forum to get support should any problems arise during installation.
