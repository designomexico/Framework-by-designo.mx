API

API Pro only

The alternative to using the HTML based approach is using the Javascript API. The API supports all the options of the HTML approach and provides some useful extra functions. For example, the most common use of the API is opening groups of images using a single link.

Fresco.show()

This function is at the core of the API, it can be used to show a single item or a group.

Fresco.show( url )

The most basic usage of Fresco.show is passing it a string. Fresco will then attempt to detect the correct way to show the file using the url.

Fresco.show( url )

url
string
A string representing the url of the file.
jQuery(document).ready(function($) {
  $('#demo_api_basic').bind('click', function(event) {
    // stop default click behavior
    event.preventDefault(); 

    Fresco.show('image.jpg');
  });
});
Show image

Fresco.show( Element )

Using an Element Object instead makes it possible to pass in more values and Options.

Fresco.show( Element )

Element
object
An Element Object, allowing for more values to be passed in than just the url.

Element = { url [ , caption ] [ , type ] [ , options ] }

url
string
A string representing the url of the file. By leaving out type the media type will automatically be detected.
caption
string
The caption underneath the content.
type
string
Specifies the media type. Required when the type cannot be detected based on the url. One of: image,youtube or vimeo
options
object
An object containing a list of Options.
jQuery(document).ready(function($) {
  $('#demo_show_element_object').bind('click', function(event) {
    // stop default click behavior
    event.preventDefault();

    Fresco.show({
      url: 'http://www.youtube.com/watch?v=c0KYU2j0TM4',
      caption: 'This video is shown using the Javascript API',
      options: {
        width: 853, 
        height: 480,
        youtube: { autoplay: 0 }
      }
    });
  });
});
Show element

Fresco.show( elements [ , options ] [ , position ] )

Groups can be shown by using an array of strings or objects, or a combination of both:

Fresco.show( elements [ , options ] [ , position ] )

elements
array
An array of Elements.
options
object
An object containing a list of Options that will be applied to all the elements. Options set on individual elements will overwrite these options.
position
number
The position to open within the elements, defaults to 1.
Fresco.show([
  'image1.jpg',
  'image2.jpg',
  { url: 'http://www.youtube.com/watch?v=c0KYU2j0TM4', options: { width: 853, height: 480 } }
]);

The second parameter can be used to provide options for the entire group.

Fresco.show(['image1.jpg', 'image2.jpg', 'image3.jpg'], {
  thumbnails: false
});
Group with grouped options

Any options defined on individual objects will overwrite what's provided using the options parameter:

Fresco.show([{
  url: 'http://www.youtube.com/watch?v=c0KYU2j0TM4', 
  options: {
    width: 853, 
    height: 480
  }
}, {
  url: 'http://www.youtube.com/watch?v=nTFEUsudhfs'
}, {
  url: 'http://www.youtube.com/watch?v=F_NEIwDiCSw'
}], {
  width: 640,
  height: 360
});
Individual & grouped options

The position at which Fresco opens can be set with the last argument. Here is an example that opens a group at the second position:

<a href="#" id="demo_api_group">Group at 2nd position</a>

<script type="text/javascript">
jQuery(document).ready(function($) {
  $('demo_api_group').bind('click', function(event) {
    // stop default click behavior
    event.preventDefault();

    Fresco.show([
      { url: 'image1.jpg' },
      { url: 'image2.jpg' },
      { url: 'image3.jpg' }
    ], { ui: 'inside' }, 2);
  });
});
</script>
Group at 2nd position

Fresco.show( elements [ , position ] )

When passing in a group of elements setting options is not required. Using a number as the last argument will then set the opening position:

Fresco.show( elements [ , position ] )

elements
array
An array of Elements.
position
number
The position to open within the elements, defaults to 1.
Fresco.show(['image1.jpg', 'image2.jpg', 'image3.jpg'], 3); // opens at 3th position
Group at 3th position, without options

Fresco.show( HTMLElement )

Fresco.show( HTMLElement )

element
HTMLElement
An HTMLElement that has the required HTML attributes set.

An HTMLElement that has the required Fresco attributes set can be shown:

Fresco.show($('#element_id')[0]);

Fresco.hide()

Hides the Fresco window when visible.

Fresco.hide();

Fresco.setDefaultSkin()

Sets the name of the default skin, this skin will be used when no skin option is provided.

Fresco.setDefaultSkin( skin )

skin
string
The name of the default skin, defaults to 'dark'.

This function should be called after the file that defines custom skins, for example:

<script type='text/javascript' src='/js/fresco/fresco.js'></script>
<script type='text/javascript' src='/js/fresco/fresco-custom-skin.js'></script>

<script type='text/javascript'>
  Fresco.setDefaultSkin('custom');
</script>
